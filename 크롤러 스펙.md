## 해커뉴스
- url: https://news.ycombinator.com
- config
 ```json
 {
   "itemSelector": "tr.athing",
   "itemRelations": {
     title: 'find(span.titleline a).text',
     url: 'next().find(span.subline a).last().attr(href)'
   },
   contentSelector: 'div.toptext'
 }
 ```
